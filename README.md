# Blood Bank Management System

## Description

This project is a console-based Blood Bank Management System written in C.
It helps manage donor and recipient records and keeps track of available blood stock in the system.

## Features

* Add new donor
* Update donor details
* Search donor by ID
* View blood stock
* Find donors by blood group
* View all donors
* Add recipient
* Manage blood stock
* Delete donor

## Technologies Used

* C Programming Language
* Structures and Arrays
* File Handling

## How to Run

1. Compile the program using a C compiler.
2. Run the main program file.
3. Use the menu to perform different operations.

## Files Used

* donors.txt – stores donor data
* recipients.txt – stores recipient data
